# Design Style Guide: Jacob Stephens Resume ## 1. Brand Personality & Tone - **Professional & Technical:** The focus is on clarity and expertise in software engineering. - **Minimalist:** A clean, no-frills aesthetic that prioritizes content over decoration. - **Accessible:** Clear hierarchy and high-contrast text ensure readability. ## 2. Visual Style - **Typography:** - **Headings:** Bold sans-serif (e.g., Arial, Helvetica, or System UI fonts). - **Body Text:** Clean sans-serif with generous line height for readability. - **Color Palette:** - **Primary Black:** `#000000` for main headings and primary text. - **Action Blue:** `#0000EE` (standard link blue) used for headers and links to denote interactivity. - **Accent Orange:** `#F3A812` for the primary "Download PDF" call-to-action button. - **Secondary Gray:** Approximately `#555555` for smaller metadata (dates, locations). - **Layout:** - **Single Column:** A straightforward, top-to-bottom flow optimized for reading. - **Whitespace:** Significant use of negative space between sections to prevent visual clutter. - **UI Elements:** - **Buttons:** Rounded corners (approx. 4px) with high-contrast text. - **Lists:** Standard bullet points for skills and work achievements. ## 3. Improvement Suggestions ### UX & Navigation - **Sticky Header:** For longer mobile views, a sticky header with the name and a "Download" shortcut would improve usability. - **Contact Icons:** Replace text-only contact info with recognizable icons (email, phone, LinkedIn) to reduce cognitive load. - **Skill Categorization:** Group technical skills into categories (e.g., "Languages," "Frameworks," "DevOps") to make the list easier to scan. ### Visual Polish - **Color Refinement:** Consider a more modern "Brand Blue" (e.g., a slightly deeper or more vibrant navy) instead of the default browser link blue. - **Visual Dividers:** Use subtle horizontal rules (`
`) or soft background shading to clearly delineate major sections like "Technical Skills" and "Work Experience." - **Typography Scaling:** Increase the contrast between section headers (e.g., "Technical Skills") and sub-headers (e.g., company names) using size or weight variations. ### Mobile Optimization - **Button Sizing:** Ensure the "Download PDF" button is large enough for easy thumb-tapping. - **Content Density:** For very long work descriptions, consider "Read More" toggles or a more condensed bullet point style to reduce excessive scrolling.
